# 💹 策略分享型：多幣種滾倉模擬平台

這是一套針對虛擬貨幣（BTC / ETH / SOL / TON / PROMP 等）所打造的合約滾倉策略模擬器，支援：

- 多幣種合約開倉模擬
- 浮盈自動加倉邏輯推演
- 成本 / 強平價 / 倉位分配圖表
- 自動獲利成長模擬（挑戰 100U ➜ 10萬U）
- 策略儲存 / 載入 / 分享 JSON 檔案
- 完整互動式網頁版本，適合手機與桌機操作

## 🧰 技術環境

- Python 3.9+
- Streamlit
- Pandas / Matplotlib / JSON IO

---

## 🧠 操作範例

1. 左側欄位輸入策略參數（幣種、槓桿、浮盈加碼邏輯）
2. 按下「📥 儲存策略 JSON」可下載設定檔
3. 日後可從「📤 載入策略」重新讀取回測
4. 中央會產生互動圖表與模擬分析
